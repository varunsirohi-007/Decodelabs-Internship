# Decodelabs-Internship
Data Analytics Project
Overview
This project demonstrates a complete Data Analytics workflow starting from raw data cleaning to analysis, SQL querying, and dashboard creation. The objective was to extract meaningful business insights from an ecommerce dataset and present them through reports and visual dashboards.

Task 1: Data Cleaning (CSV)
In this phase, the raw dataset was cleaned and prepared for analysis.

Steps Performed
Imported raw dataset into Excel/Python
Removed duplicate records
Handled missing/null values
Standardized column names and formats
Checked data consistency
Exported final cleaned dataset in CSV/Excel format
Output
Cleaned Dataset (Cleaned_Dataset.xlsx / CSV)

Task 2: SQL Data Analysis
Used SQL queries to analyze the dataset and answer business questions.

SQL Operations Performed
Created database table schema
Used SELECT queries
Applied WHERE conditions
Used GROUP BY and ORDER BY
Aggregate functions:
COUNT()
SUM()
AVG()
MAX()
MIN()
Business Queries
Total Revenue
Total Orders
Top Selling Products
Monthly Sales Analysis
Payment Method Distribution
Coupon Usage Analysis
Referral Source Performance
Output
SQL Queries File (SQL_Queries.sql)

Project Outcome
This project demonstrates an end-to-end Data Analytics process, including:

Data Cleaning
SQL-based Data Querying
The project helped in understanding how raw data can be transformed into meaningful business insights for decision making.
