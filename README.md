# Decodelabs-Internship
Data Analytics Project
Overview
This project demonstrates a complete Data Analytics workflow starting from raw data cleaning to analysis, SQL querying, and dashboard creation. The objective was to extract meaningful business insights from an ecommerce dataset and present them through reports and visual dashboards.

Task 1: Data Cleaning (CSV)
In this phase, the raw dataset was cleaned and prepared for analysis.

Steps Performed
Imported raw dataset into Excel/Python
Removed duplicate records
Handled missing/null values
Standardized column names and formats
Checked data consistency
Exported final cleaned dataset in CSV/Excel format
Output
Cleaned Dataset (Cleaned_Dataset.xlsx / CSV)
Task 2: Exploratory Data Analysis (EDA)
Performed analysis to identify patterns, trends, and useful business insights.

Analysis Performed
Descriptive statistics (Mean, Median, Count, Min, Max)
Missing value analysis
Outlier detection
Product sales trend analysis
Revenue analysis over time
Payment method analysis
Referral source performance analysis
Order status distribution
Correlation analysis between numeric columns
Output
EDA Report (EDA_Report.pdf)
Python analysis code
Task 3: SQL Data Analysis
Used SQL queries to analyze the dataset and answer business questions.

SQL Operations Performed
Created database table schema
Used SELECT queries
Applied WHERE conditions
Used GROUP BY and ORDER BY
Aggregate functions:
COUNT()
SUM()
AVG()
MAX()
MIN()
Business Queries
Total Revenue
Total Orders
Top Selling Products
Monthly Sales Analysis
Payment Method Distribution
Coupon Usage Analysis
Referral Source Performance
Output
SQL Queries File (SQL_Queries.sql)
Task 4: Dashboard Development
Built an interactive dashboard using Microsoft Power BI.

Dashboard Features
Sales Dashboard
Total Revenue KPI
Total Orders KPI
Average Order Value
Monthly Revenue Trend
Product Dashboard
Product-wise Sales Analysis
Quantity Sold by Product
Customer & Marketing Dashboard
Payment Method Analysis
Coupon Usage Analysis
Referral Source Analysis
Operations Dashboard
Order Status Distribution
Items in Cart Analysis
Filters/Slicers for Date and Product
Output
Power BI Dashboard
DAX Measures File (PowerBI_DAX.txt)
Tools Used
Microsoft Excel
Python (Pandas, Matplotlib)
SQL
Microsoft Power BI
Project Outcome
This project demonstrates an end-to-end Data Analytics process, including:

Data Cleaning
Exploratory Data Analysis
SQL-based Data Querying
Business Intelligence Dashboard Creation
The project helped in understanding how raw data can be transformed into meaningful business insights for decision making.
