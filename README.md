# 📊 DecodoLabs Internship – Data Analytics Project

## 📌 Project Overview

This project demonstrates an end-to-end **Data Analytics workflow**, starting from raw data cleaning and preparation to SQL-based analysis and business insight generation. The goal was to transform raw e-commerce data into meaningful information that supports data-driven decision-making.

---

## 🛠️ Tools & Technologies Used

* **Python (Pandas)** – Data cleaning and preprocessing
* **Excel** – Initial data inspection and validation
* **SQL** – Data querying and business analysis
* **CSV/Excel Files** – Data storage and management

# 🧹 Task 1: Data Cleaning

The raw dataset contained inconsistencies, missing values, and duplicate records. Data cleaning was performed to improve data quality and ensure accurate analysis.

### Steps Performed

* Imported raw dataset into Python and Excel
* Removed duplicate records
* Handled missing/null values
* Standardized column names and formats
* Checked data consistency and accuracy
* Exported the final cleaned dataset

### Output

* `Cleaned_Dataset.csv`
* `Cleaned_Dataset.xlsx`

---

# 🗄️ Task 2: SQL Data Analysis

After cleaning the data, SQL queries were used to extract business insights and answer analytical questions.

### SQL Concepts Applied

* SELECT Statements
* WHERE Clauses
* GROUP BY
* ORDER BY
* Aggregate Functions:

  * COUNT()
  * SUM()
  * AVG()
  * MAX()
  * MIN()

### Business Questions Solved

* Total Revenue Generated
* Total Orders Placed
* Top Selling Products
* Monthly Sales Trends
* Payment Method Distribution
* Coupon Usage Analysis
* Referral Source Performance

### Output

* `SQL_Queries.sql`

---

# 📈 Key Insights Generated

* Identified revenue trends across different periods
* Analyzed customer purchasing behavior
* Determined the most popular products
* Evaluated effectiveness of coupon campaigns
* Compared performance of referral channels
* Examined preferred payment methods
